# 🔐 BloodyKey — безопасное шифрование в браузере (AES-256-GCM)

**BloodyKey** — простой и надёжный инструмент для шифрования текста и файлов.  
Вся магия происходит прямо в вашем браузере — данные никуда не отправляются.

## ✨ Что умеет?

- **AES-256-GCM** — промышленный стандарт шифрования (NIST).
- **PBKDF2** с 100 000 итерациями SHA‑256 для превращения пароля в ключ.
- **Два режима**:
  - 📝 **Текст** → результат в **Base64** (удобно для мессенджеров, почты).
  - 📁 **Файл** → компактный **.bloody** файл (размер увеличивается всего на 28 байт).
- **Проверка сложности пароля** (цветовая шкала).
- **Drag & drop** для файлов.
- **Прогресс-бар** при обработке.
- **Адаптивный дизайн** (тёмная тема, стеклянный эффект).
- **Горячая клавиша** `Ctrl+E` (зашифровать / расшифровать).

## ⚙️ Как это работает внутри?

1. Вы вводите пароль — он **не сохраняется** и **не передаётся**.
2. Генерируется случайная **соль** (16 байт) и **вектор инициализации IV** (12 байт).
3. Пароль + соль → **PBKDF2** → 256-битный ключ (100 000 итераций).
4. Данные шифруются **AES-256-GCM**, который также добавляет **тег аутентификации** (защита от подмены).
5. Результат:
   - для текста — одна строка **Base64** (содержит соль, IV и шифротекст);
   - для файла — бинарный **.bloody** файл (соль + IV + шифротекст).

Расшифровать без пароля невозможно — даже автор не поможет.

## 🚀 Как запустить?

### Локально
- Скачайте `index.html`.
- Откройте в **современном браузере** (Chrome, Firefox, Edge, Safari).
- **Важно:** из-за требований Web Crypto API используйте **HTTPS** или `localhost` (простой локальный сервер: `npx serve .`). Если открыть файл через `file://`, браузер может заблокировать криптографию.

## 📖 Как пользоваться

### 🔒 Шифрование текста
1. Выберите **«Зашифровать» → «Текст»**.
2. Введите пароль (минимум 12 символов).
3. Напишите сообщение.
4. Нажмите **«Зашифровать»** → скопируйте полученную Base64-строку.
5. Отправьте строку другу (любым каналом).

### 🔓 Расшифровка текста
1. **«Расшифровать» → «Текст»**.
2. Вставьте Base64-строку.
3. Введите тот же пароль → нажмите **«Расшифровать»**.

### 📁 Шифрование файла
1. **«Зашифровать» → «Файл»**.
2. Нажмите на область или перетащите файл.
3. Введите пароль → **«Зашифровать»** → скачается файл `имя.bloody`.
4. Передайте `.bloody` файл получателю.

### 📂 Расшифровка файла
1. **«Расшифровать» → «Файл»**.
2. Выберите `.bloody` файл.
3. Введите пароль → **«Расшифровать»** → исходный файл восстановится с оригинальным именем.

## ⚠️ Важные предупреждения

- **Пароль нельзя передавать по тому же каналу, что и зашифрованные данные.** Договоритесь лично, по телефону или через защищённый мессенджер (Signal, Telegram secret chat).
- **Нет функции восстановления пароля.** Забудете пароль — данные потеряны навсегда.
- Всё шифрование происходит **на вашем устройстве**. Автор не имеет доступа к вашим данным.
- Инструмент предназначен для **личного использования**. Для особо ценных данных используйте сертифицированное ПО.
